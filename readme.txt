Turns off the Gutenberg content editor in WordPress 5.0+

== Description ==

If your site is not yet ready for the Gutenberg editor in WordPress 5.0+ install this plugin to disable the new functionality

### Why not use the existing Classic Editor plugin?

The [Classic Editor](https://wordpress.org/plugins/classic-editor/) has a multiple step setup/configuration for older WordPress installs

1. Install and activate the Classic Editor plugin
2. Prior to WordPress 5.0 activate the Gutenberg plugin
3. Go to Settings > Writing to uncheck the default "Do not replace the editor"

Activate this plugin to ensure Gutenberg goes away; simply deactivate this plugin when you're ready for Gutenberg
